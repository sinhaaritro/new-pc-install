# Microsoft Office Installation (Windows)

To download and activate Microsoft Office, we recommend using the Microsoft Activation Scripts (MAS) repository.

1. **Visit the official MAS guide**:
   Navigate to [https://github.com/elitekamrul/MAS](https://github.com/elitekamrul/MAS) or the official Microsoft Activation Scripts repository.
2. Follow the instructions provided in the repository to download, install, and securely activate the appropriate version of Office.
