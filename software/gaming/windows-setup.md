# Windows Gaming Setup

1. Download and install **Steam** and the **Epic Games Launcher**.
2. Log in and allow the apps to finish their background updates before installing games.
3. **Discord Multi-Monitor Tweak:**
   * Open Discord, ensure it's **NOT maximized**.
   * Drag the small Discord window to your desired secondary monitor.
   * **Close Discord** while it's still small and on that monitor.
   * Re-open Discord. It should now default to opening on that screen! You can maximize it normally afterwards.
